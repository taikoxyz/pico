# @tainnel/wallet-ui

Reference web wallet for tainnel payment channels. React 18 + Vite + TypeScript, viem
for crypto, wagmi v2 for connector management, Tailwind + a shadcn/ui-style component
layout. Connects to one or more hubs over WebSocket using `@tainnel/sdk`.

Pages: dashboard, open-channel flow, manual pay, DVM browser, settings.

```bash
pnpm --filter @tainnel/wallet-ui dev
```
